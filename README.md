# ✨ 팬 커머스샵 VeryPeri


## 🔎프로젝트 개요

📆 2023.08.09 ~ 2023.08.28 (22일)<br>
더조은컴퓨터아카데미 수강 첫 번째 팀 프로젝트.
<br>
<br>
## 🔎프로젝트 소개
현대의 소비 문화는 끊임없이 변화하고 있으며, 젊은 대상층은 새로운 경험과 소비 방식을 찾고 있습니다.<br> VERY PERI는 이러한 변화를 반영하며, 젊은 소비층을 위한 혁신적인 팬 커머스 샵 프로젝트입니다.
<br>

## 💁‍♂️ 담당부분
더미데이터 삽입  <br>
상품 페이지 프론트 <br>
DB 데이터 입력  <br>
마이페이지 전체 프론트 <br>
고객 포인트 사용 내역 <br>
장바구니 프론트 <br>
결제페이지 프론트 <br>
관리자 상품 등록 이미지 첨부 구축 <br>
헤더 푸터 프론트 및 전체 화면 인클루드 <br>
회원정보 수정의 프로필 변경 <br>

<br>
<br>
